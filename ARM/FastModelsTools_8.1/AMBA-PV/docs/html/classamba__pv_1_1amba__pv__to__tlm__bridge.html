template<unsigned int BUSWIDTH = 64>
class amba_pv::amba_pv_to_tlm_bridge< BUSWIDTH >

AMBA-PV to OSCI TLM 2.0 BP bridge module.

The amba_pv_to_tlm_bridge class converts AMBA-PV transactions into OSCI TLM 2.0 BP transactions.

Blocking transport.

This version of the method converts an AMBA-PV transaction into an OSCI TLM 2.0 BP transaction. The main change is conversion of wrapping bursts into incremental burts. In addition, the transaction address is aligned to the burst size.

Implements amba_pv::amba_pv_fw_transport_if.
