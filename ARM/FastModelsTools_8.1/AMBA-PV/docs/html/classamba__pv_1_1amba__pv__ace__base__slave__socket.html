Detailed Description

template<unsigned int BUSWIDTH = 64, int N = 1, sc_core::sc_port_policy POL = sc_core::SC_ONE_OR_MORE_BOUND>
class amba_pv::amba_pv_ace_base_slave_socket< BUSWIDTH, N, POL >

AMBA-PV base slave socket.

This socket inherits from the tlmx::tlmx_target_socket full-duplex socket class and implements a tagged socket. A tagged socket allows a component to determine through which socket an incoming method call arrived. This is required if there are multiple slave sockets such as in, for example, a bus decoder or a multi-port memory.

Parameters:
BUSWIDTH bus width in bits as one of 8, 16, 32, 64, 128, 256, 512, or 1024. Defaults to 64.
N number of bindings. Defaults to 1.
POL port binding policy. Defaults to sc_core::SC_ONE_OR_MORE_BOUND.
